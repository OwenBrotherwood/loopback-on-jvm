# LoopBack on JVM

## Set up

1. Download and install JDK 8 Build b108

https://jdk8.java.net/download.html

2. Check out avatar-js source code and related build tools


    ./checkout.sh

3. Build avatar-js


    ./build.sh

## Run samples

    nodej hello

## Run LoopBack sample app

    cd sls-sample-app
    npm install
    nodej app
